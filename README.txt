Программа написана на языке Python.
Исходный код в файле Programm_for_CSV.txt
Для примера работы программы используется файл data_users.csv
Programm_for_CSV.exe - программа, предназначенная для работы с файлами формата CSV.
CSV(Comma-Separated Values - значения, разделенные запятыми) - текстовый формат,
предназначенный для представления табличных данных. Строка таблицы соответствует 
строке текста, которая содержит одно или несколько полей, разделенных запятыми.
Пример:
hostname,vendor,model,location
sw1,Cisco,3660,"London, West str"
sw2,Cisco,3850,"Liverpool, Better str"
sw3,Cisco,3650,"Liverpool, Better str"
sw4,Cisco,3650,"London, Best str"
***********************************************************************************
***********************************************************************************
Запуск программы
В папкe Programm_for_CSV запускаем файл Programm_for_CSV.exe
***********************************************************************************
Работа с программой
	При запуске необходимо ввести путь к файлу в формате CSV :
	Пример:D:\data_about_users_new.csv
Далее переход в главное меню.
	При выборе 'Вывод данных о пользователях', выводится информация о пользователях
в виде таблицы.
***********************************************************************************
	При выборе 'Для выхода из программы', осуществляется выход из программы.
***********************************************************************************	
	При выборе 'Изменение пользователей', осуществляется переход в подменю:
	При каждом изменении файла, необходимо вводить имя и путь для сохранения файла.
	'Удаление пользователей' - удаление необходимого кол-ва пользователей.
		далее переходим к 'Сколько пользователей удалить?' - вводим кол-во
пользователей для удаления.
		далее переходим к 'Удалить пользователя под номером:'.
Номера присвоиваются таким образом:
   	hostname,vendor,model,location
   1	sw1,Cisco,3660,"London, West str"
   2 	sw2,Cisco,3850,"Liverpool, Better str"
   3 	sw3,Cisco,3650,"Liverpool, Better str"
   4	sw4,Cisco,3650,"London, Best str"
К примеру, удаляем пользователя под с 'hostname' sw1, его номер 1. После этого удаления список преобразуется
к следующему виду:
   	hostname,vendor,model,location
   1 	sw2,Cisco,3850,"Liverpool, Better str"
   2 	sw3,Cisco,3650,"Liverpool, Better str"
   3	sw4,Cisco,3650,"London, Best str"
Теперь y пользователя sw2 номер 1. Далее при удалении пользователя sw4, вводим 3 и так далее.
	'Добавление пользователей' - происходит добавление в файл данных о новых пользователях.
		далее переходим к 'Сколько пользователей ввести?' - ввод кол-ва пользователей.
		далее вводим имя; 
		аппаратуру, через которую работает пользователь;
		модель;
		месторасположение.
	'Изменение данных о пользователях' - изменение определенных данных о пользователя.
	'Скольким пользователям изменить данные' - вводим кол-во пользователей, которым необходимо
изменить. 
	'Сколько параметров пользователя необходимо изменить' - вводим кол-во параметров пользователей, которые необходимо
изменить.
	Далее выбираем те параметры, которые необходимо изменить. 
	После изменения данных вводим путь для сохранения файла и название файла
	После сохранения файла появляется 'Нажмите ENTER для выхода в главное меню.'
